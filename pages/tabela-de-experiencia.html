HTML sigla para "HyperText Markup Language", trata-se de uma linguagem de marcação utilizada para criar e estruturar o conteúdo das páginas da internet. Combinado com outras tecnologias como CSS e JavaScript, o HTML permite criar páginas estáticas e dinâmicas. 1 ano Intermediário
CSS "Cascading Style Sheets", é uma linguagem de estilo utilizada em conjunto com o HTML para controlar a apresentação e o layout das páginas da web. Ele define a aparência dos elementos HTML, permitindo ajustar cores, fontes, tamanhos, margens e outros aspectos visuais. 7 meses Iniciante
